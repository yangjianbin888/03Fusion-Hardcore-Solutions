**1. 痛点定义（Why）**

现有60分方案的失效机理：
- **表面转化效率低**：H⁻/D⁻主要靠Cs表面转化（低功函数），但Cs层分布不均→局部功函数波动→D⁻产额不均匀，体积产生占比低（<15%），整体提取效率卡在35~45%。
- **磁过滤场致束偏转与不均匀**：B_filter（~数十mT横向）滤除热电子但同时造成等离子体鞘层畸变、D⁻轨道偏折→束剖面非均匀，边缘损失大。
- **大口径多孔栅极边缘效应**：大面积提取栅（数百至千孔）边缘电场跌落→边缘束流密度低→均匀性劣化至±15%以上。
- **共引出电子负载**：电子被同步加速→中和器/抑制极功耗剧增，限制占空比与功率密度。
- **热变形漂移栅位形**：MW级束功率下栅极热翘曲→光学像差，长期运行均匀性退化。

**2. 破局方案（What）**

**核心架构**：大面积分段RF驱动Cs表面转化源 + 分布式主动Cs涂层闭环控制 + 渐变补偿磁过滤场 + 多级孔径梯度电压预加速栅（PG＋EG＋G1～G3） + 横向E×B电子偏转抑制 + 水冷等刚度栅框架——反共识点在于**放弃"均一磁过滤+被动Cs涂覆"传统构型，改用分段磁过滤+主动Cs通量反馈+边缘电场提升环(Edge Enhancement Ring)**来实现>60%提取效率和±5%均匀性。

- 提取/加速栅极：需满足 OFHC 无氧铜（Cu-OFE，ASTM F68）或 Mo（ASTM B387 Grade 361），厚度公差 ±0.02mm，孔径 φ2–4mm，孔间距 pitch 依均匀性仿真确定，热变形 < 50μm @ 满功率。
- 磁过滤与偏转磁铁：需满足 NdFeB N42 或水冷铜电磁线圈，产生 B⊥=20–80mT 可调，磁场均匀性各段 ±5%。

**3. 实施路径（How）**

**步骤 A：[大面积分段RF源+主动Cs分布控制样机建造与等离子体诊断]**
- 动作：建造 1/2 尺寸（~0.2×0.4m 提取面）RF源，布置多点 Langmuir 探针 + 发射光谱 Cs* 线强度成像，闭环调节各段 Cs 蒸发速率维持表面功函数 φ < 1.6eV 均匀性 ±0.05eV。
- 验收标准：D⁻体密度 > 5×10¹⁶ m⁻³，Cs* 发射空间非均匀性 < ±10%，D⁻/e⁻ 比 > 2（未加速）。

**步骤 B：[渐变补偿磁过滤+边缘增强栅光学调试]**
- 动作：磁过滤场沿束轴分段递减（入口 B_max→出口 B_min），抵消 E×B 偏折；提取栅外加同心 Edge Enhancement Ring 施加 +5%~10% 相对 Gnd 偏置提升边缘电场；PIC 束光学仿真→实测比对。
- 验收标准：单孔 D⁻ 提取效率 > 55%（局部），大面积束剖面均匀性 ≤ ±7%（RMS），电子引出的横向偏转分离率 > 80%。

**步骤 C：[全尺寸兆瓦束源集成+加速测试+量产放行]**
- 动作：拼合多驱动模块至全尺寸提取面（~0.4×1.0m 或等效 1000+ 孔），三级加速至 500keV，束_dump 量热+光栏扫描测均匀性，长期热循环考核。
- 验收标准：D⁻总束功率 ≥1MW @ 500keV，提取效率 ≥60%，均匀性 ≤±5%（RMS），电子抑制后 e⁻/D⁻ < 0.3，连续运行 ≥300s 可重复 >10 次，栅热变形回零（冷却后）。
